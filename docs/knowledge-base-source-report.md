## 法律规则与内部运营规则的区别

每篇政策文档中以下标记表示内部运营规则：

```
## ResolveAI 内部运营规则
> 以下为平台运营规则，非法律规定。
```

- 审批阈值（如 ¥1,000 高额退款阈值）是内部运营规则，不来自法律条文
- 升级流程和人工审核触发条件是内部运营规则
- 换货品类特殊规则以法律、规章和司法解释为依据；具体审核、差价和运费流程属于 ResolveAI 内部规则
- 包装拆封本身不等于商品不完好；合理查验和调试不影响商品完好
- 电子产品、耳机、食品等不得仅按大类擅自扩大不适用退货范围
