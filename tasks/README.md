# Tasks — Phase Tracking

This directory tracks the development progress of ResolveAI across sequential phases.

## Structure

- `active-phase.md` — Points to the currently active phase. Read this first.
- `backlog.md` — Ideas and features deferred to future versions.
- `phase-NN-*.md` — One file per development phase with goals, tasks, and acceptance criteria.

## How to Use

1. Read `active-phase.md` to know the current phase.
2. Open the current phase file for detailed tasks.
3. Check `backlog.md` for future ideas.
4. Each phase must be completed and verified before moving to the next.
