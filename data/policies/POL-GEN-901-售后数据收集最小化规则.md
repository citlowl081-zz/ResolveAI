# 售后数据收集与个人信息保护

## 数据最小化原则
1. 仅收集为完成售后所必需的最少个人信息；
2. 售后退款、补发等操作不要求消费者提供超过必要范围的证件；
3. 消费者提供的证据照片仅用于售后审核，并按公开的保存期限和删除规则处理。

## 禁止收集的信息
1. 完整的银行卡号或支付密码；
2. 身份证完整照片或号码（身份核验除外）；
3. 与售后无关的家庭成员信息；
4. 消费者的精确家庭地址（物流地址除外）。

## 支付信息安全
> 以下支付处理方式为 ResolveAI 内部安全规则，不是个人信息保护法的具体操作要求。

1. 退款操作通过支付平台完成，平台不直接处理银行卡信息；
2. 退款原路返回至消费者支付账户；
3. 任何售后环节不得要求消费者提供支付密码。

## AI Agent Memory 限制
1. AI Agent 不得将消费者的身份证号、银行卡号、密码等敏感信息写入长期记忆；
2. 系统在写入 Memory 前自动进行敏感信息检测；
3. 检测到敏感信息时拒绝写入并记录安全日志。

## 重要声明
ResolveAI 知识库中的政策规则不等于法律建议。正式商用前应由法务进行合规审查。
